--How to Subnet IPv4 Addresses--

-Subnetting a Class C network-
Say you have a network address of 192.168.100.0/24 and you need nine subnets.
You can only use H bits, not N bits. So you can ignore the 192.168.100 part.
	Remember for Class C networks that it's like such:

	|Octet 1		|Octet 2		|Octet 3		|Octet 4	|
	|NNNNNNNN		|NNNNNNNN		|NNNNNNNN		|HHHHHHHH	|
	
	You will need to borrow some host bits and turn them into network bits (Or in this case, subnet bits. I will use S to show that it is not a subnet bit)
	
	First you need to know how many H bits you need to borrow. 
	2^N is greater than or equal to 9 (It doesn't let me use the symbol, it's ">" with a line under)
	N is equal to 4 so 4 H bits
		|Start with 8 H bits	|HHHHHHHH
		|Borrow 4 bits			|SSSSHHHH
		
	Determine the first subnet in binary
		|0000HHHH		|
		|00000000		|All 0s in host portion = subnetwork number
		|00000001		|First valid host number
		|00000010		|Second valid host number
		|00000011		|Third vaild host number
		(This patter continues for rest of host numbers)
		|00001110		|Last valid host number
		|00001111		|All 1s in host portion = broadcast number
		
		Convert binary to decimal
			|Binary		|Decimal	|Notes
			|00000000	|0			|Subnet number
			|00000001	|1			|First valid host
			|00000010	|2			|Second valid host
			|00000011	|3			|Third valid host
			(Pattern once again continues)
			|00001110	|14			|Last valid host
			|00001111	|15			|Boardcast number
			
	Determine the second subnet in binary
		|0001HHHH	|
		|00010000	|subnet number
		|00010001	|First valid host
		(You get the idea)
		|00011110	|Last valid host
		|00011111	|Boardcast number
		
		Convert binary to decimal
		|Binary		|Decimal	|Notes
		|00010000	|16			|Subnet number
		|00010001	|17			|First valid host
		|00010010	|18			|Second valid host
		(Please tell me you get the idea by now)
		|00011110	|30			|Last valid host
		|00011111	|31			|Boardcast number
		
	Build IP plan table
	|Subnet	|Network number	|Range of valid hosts	|Boardcast number	|
	|1		|0				|1-14					|15					|
	|2		|16				|17-30					|31					|
	|3		|32				|33-46					|47					|
	See a pattern, counting by 16

	Calculate the subnet mask
	|Orginal mask	|11111111.11111111.11111111.00000000	|255.255.255.0	 |
